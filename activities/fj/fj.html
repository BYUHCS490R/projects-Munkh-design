What is AJAX?
Why do you put your JS Code in a separate file?
AJAX is a technique used in web to request data from a server and update parts of the page without needing a full page reload.
Putting JavaScript code in a separate file improves organization, reusability, caching performance, separation of concerns, debugging, and security.
